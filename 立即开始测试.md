# 🚀 立即开始测试

## ⚡ 3 分钟快速开始

### 第 1 步：初始化配置（1 分钟）

#### 访问初始化页面
```
http://localhost:3000/admin/initialize
```

#### 操作
1. 点击右上角连接 Solana 钱包
2. 点击 "Check Configuration Status"
3. 如果显示 "Config not found"：
   - 点击 "Initialize Configuration"
   - 在钱包中批准交易
   - 等待确认（约 2-3 秒）

#### 验证
看到 "✅ Config exists!" 表示成功

---

### 第 2 步：获取测试代币（1 分钟）

#### 获取 SOL
```
https://faucet.solana.com/
```
- 输入钱包地址
- 选择 Devnet
- 请求 2 SOL

#### 获取 USDC
```
https://spl-token-faucet.com/?token-name=USDC
```
- 输入钱包地址
- 请求 1000 USDC

---

### 第 3 步：开始测试（1 分钟）

#### 创建市场
```
http://localhost:3000/markets/create
```

填写：
- 问题: "Will Bitcoin reach $100k?"
- YES 符号: "YES"
- YES URI: "https://example.com/yes.json"

点击 "Create Market"

## ✅ 完整测试流程

### 1️⃣ 初始化配置
```
http://localhost:3000/admin/initialize
```
- 连接钱包
- 初始化配置
- ✅ 完成

### 2️⃣ 创建市场
```
http://localhost:3000/markets/create
```
- 填写市场信息
- 创建市场
- ✅ 完成

### 3️⃣ 添加流动性
在市场详情页：
- 切换到 "Liquidity" 标签
- 输入 100 USDC
- 点击 "Add Liquidity"
- ✅ 完成

### 4️⃣ 买入 YES 代币
在市场详情页：
- 切换到 "Trade" 标签
- 选择 "Buy YES"
- 输入 10 USDC
- 点击 "Buy"
- ✅ 完成

### 5️⃣ 卖出 YES 代币
- 选择 "Sell YES"
- 输入 5 YES
- 点击 "Sell"
- ✅ 完成

### 6️⃣ 买入 NO 代币
- 选择 "Buy NO"
- 输入 10 USDC
- 点击 "Buy"
- ✅ 完成

### 7️⃣ 卖出 NO 代币
- 选择 "Sell NO"
- 输入 5 NO
- 点击 "Sell"
- ✅ 完成

### 8️⃣ 移除流动性
- 切换到 "Liquidity" 标签
- 输入 LP 份额数量
- 点击 "Remove Liquidity"
- ✅ 完成
