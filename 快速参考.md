## 🐛 常见错误速查

| 错误 | 解决方案 |
|------|----------|
| Insufficient balance | 访问水龙头获取 SOL |
| Wallet not connected | 点击右上角连接钱包 |
| Transaction rejected | 在钱包中点击"批准" |
| Network error | 检查钱包是否在 Devnet |
